# Upload plugin

## Plugin route

Toggles the availability of the `/plugin` route.

| Value    | Effect               |
| -------- | -------------------- |
| Enabled  | Enable plugin route  |
| Disabled | Disable plugin route |

## PUP SDK URL

Determines the URL for PUP SDK.

| Type   | Accepted |
| ------ | -------- |
| String | URL      |
